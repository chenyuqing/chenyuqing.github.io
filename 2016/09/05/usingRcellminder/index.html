Using rcellminer

Overview

  • NCI-60癌症细胞系在过去的几十年中被用作研究抗癌药物帅选的课题。这个细胞系是由美国国家癌症机构[NCI]发展的Developmental Therapeutics Program, DTP 项目的一部分。成千上万的化合物[药物]已经在NCI-60上进行了测试，NCI-60通过多个平台集成了基因和蛋白质表达式，复制数，突变或其他(Reinhold, et al., 2012)。Cellminer 项目 的主要目的是从不同的平台把数据集成，用作分析，然后提供了一个对NCI-60数据分析的强有力的工具集。由于Cellminer还不是一个完整的在线探索NCI-60数据的资源，所以有很多特定的科学问题通常都需要自定义编程。rcellminer这个R包补充了Cellminer的功能，包括编程数据的接入和数据的分析和可视化。这些功能对于R的初学者来说也是合适的，譬如下面的第一个例子。取数据的子集，受Cellminer相关的出版物所启发，展示了如何用简洁的代码编写指定的分析，通过分析多种类型的数据得到一个新的科学视野。rcellminer还为拓展工具提供了坚实的构建模块，譬如最后的交互的Shiny程序。

查找药物

  • NSC数字是一个数字化的标志符，它代表着一种提交到NCI美国国家癌症机构作测试和评估的药物实例。它是一个Developmental Therapeutics Program (DTP) **资源库的注册号码，是用来唯一确定Cellminer数据库中的药物。NSC**代表National Service Center.
  • rcellminer允许用户通过药物的全名或名字的一部分查找NSC IDs。譬如，很多激酶抑制剂的名字都是以“nib”结尾的。用户能够通过这个后缀快速查找相关的NSCs。查找不分大小写并且以正则表达式的形式读取的。
