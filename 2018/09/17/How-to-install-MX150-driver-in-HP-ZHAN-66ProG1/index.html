How to install MX150 driver in HP ZHAN 66ProG1

2018-09-17
  1. Install the Ubuntu system(Recommand 16.04) and select your best server and make your system updated.
  2. reboot and turn into the BIOS setup and make the security boot disable.
  3. blacklist the nouveau
    (1) sudo gedit /etc/modprobe.d/blacklist.conf
    (2) add the follow blacklists in the end:
    blacklist vga16fb
    blacklist nouveau
    blacklist rivafb
    blacklist rivatv
    blacklist nvidiafb
    exit and saved.
  4. Remove the original nvidia driver
    sudo apt-get –purge remove nvidia-*
    sudo apt-get –purge remvoe xserver-xorg-video-nouveau
  5. Press Alt+Ctrl+F1 into the control terminal and login your root account
    sudo service lightdm stop
  6. Add the PPA of driver into system:
    sudo add-repository ppa:graphics-drivers
    sudo apt-get update
  7. Search the newest version of driver
    sudo apt-cache search nvidia
  8. Install the newest version of driver(example, nvidia-384)
    sudo apt-get install nvidia-384
  9. After installation complete, reboot
  10. Verify the installation with : lsmod | grep nvidia, it will give you some results
  11. lsmod | grep nouveau, it shows nothing to you.
  12. It’s optional that you can hold the version of driver: sudo apt-mark hold nvidia-384
