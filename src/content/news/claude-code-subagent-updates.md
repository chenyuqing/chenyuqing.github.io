**事件：** Anthropic 在 6 月 24-27 日密集发布 Claude Code 三个版本更新：v2.1.191 新增 `/rewind` 命令、降低 37% 流式响应 CPU 占用；v2.1.193 新增自动模式分类路由与后台内存压力回收；v2.1.195 修复多语言语音输入并增强后台 Agent 可靠性。子 Agent 已支持最多 5 层深度嵌套调用。

**关注原因：** Claude Code 持续深化 Agent 能力——从单 Agent 编程助手演进为多 Agent 协作系统，子 Agent 深度嵌套能力标志着 AI coding 工具正向"工程级多 Agent 编排"快速进化。
