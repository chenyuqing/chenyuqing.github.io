**事件：** 日本 Sakana AI 发布 Fugu，将多智能体编排封装为单一 API 对外售卖。Fugu Ultra 通过一个经特殊训练的模型学习智能体协作逻辑——简单问题自行处理，复杂问题自动组建专家模型团队，对用户完全透明。在工程、科学和推理基准上，Fugu Ultra 表现媲美 Claude Fable 5 和 Mythos Preview。

**关注原因：** 这是首个明确将"多智能体编排"作为服务商品化的产品，且模型池完全可替换以规避出口管制，直接威胁 Anthropic/OpenAI/Google 等模型 API 直销模式。
