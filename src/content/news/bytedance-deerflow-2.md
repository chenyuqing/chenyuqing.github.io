**事件：** 字节跳动发布 DeerFlow 2.0 开源框架，专为需要长时间连续运行的"超级 Agent"设计，集成沙箱隔离、高级记忆系统、子 Agent 编排和消息网关等能力，支持研究、编程和创意生产等复杂长任务场景。

**关注原因：** 这是开源社区首个系统化解决 Agent 长期任务执行问题的框架，其子 Agent 编排和持久化记忆架构可能成为后续 Agent 框架的参考范式。
