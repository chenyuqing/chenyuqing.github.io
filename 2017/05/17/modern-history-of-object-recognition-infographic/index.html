重要的卷积神经网络(CNN)概念

  • 特征(Feature)(模式，神经元的激活，特征检测器)

    • 当一个特定的模式(特征)在它的输入区域(receptive field)中展现时,一个隐藏的神经元就会被激活
    • 能被神经元检测的模式看通过一下方式可视化：(1) 优化它的输入区域使神经元的激活最大化(deep dream)，(2) 可视化输入像素中的神经元激活的梯度或者guided梯度(后向传播或者guided后向传播),(3) 可视化训练数据集当中最常激活神经元的一个图像区域集。

  • 接受域/感受野(Receptive Field)(一个特征的输入区域(input region))

    • 输入图像当中响应特征激活的区域，也就说，特征正在看的区域
    • 一般的，高层layer当中的特征会包含更多的接受域，它可以去学习捕获更复杂或者更抽象的模式。ConvNet架构决定了如何一层一层地改变接受域的。

  • 特征图谱(Feature Map)(隐藏层的一个channel)

    • 通过相同的特征检测器(feature detector)在输入图谱中不同的位置以滑动窗口的方式获得的一个特征集合(比如，卷积)。相同的特征图片当中的特征有相同的接受的大小，它们都是在不同的位置上寻找相同的模式。这就创造了ConvNet的空间不变性的特性。

  • 特征量(Feature Volume)(一个ConvNet的隐层)

    • 一个特征图谱(Feature Map)的集合，每个特征图普在输入图像上的设定的位置集合去寻找一个特定的特征。所有的特征有相同的receptive field size.

  • 全链接层作为特征量(Feature Volume)

    • K个节点的全链接层,(通常是加在ConvNet的最后作为一个分类器)，可以被看作是1x1xK的特征量(feature volume)。这个特征量(feature volume)在每一个特征图谱当中只有一个特征，而且它的接受域覆盖了整张图像。全连接层中的权重矩阵W可以被转换成一个CNN kernel。对一个w*h*k的kernel进行卷积，变成了CNN特征量(feature volume) w*h*d ，创建了1*1*k 特征量(等于k个节点的全连接层)。对一个1*1*k 的过滤kernel进行卷积，变成了1*1*d的特征量，创建了一个1*1*k 特征量。通过卷积层替代全连接层允许我们把ConvNet应用到任何大小的图像上。

  • 转置卷积(分部跨步卷积，去卷积，下采样)

    • 向后传播卷积操作的梯度的这个操作。换句话说，它是卷积层的反向传递。转置卷积可以这样实现：在输入特征之间插入0+做一个正常的卷积。
