how to install Google-pinyin on Ubuntu

2017-02-28
  • Ubuntu

  • Recently I have been tried used the Ubuntu for a while. And I have re-installed system for so many times. And I found that it’s so much trouble for me to install the Chinese input method. Because I have to search the Internet every time. So now I want to conclude how to install a Chinese Input Method simply. Google-pinyin for example.

  • First, you have to install the Google-pinyin package

  • Then, come to the configure

  • At this step you will get a series of choice to choose. You should choose the “yes” if it’s one-out-of-two choice. And at the middle you have to choose the “fcitx” input frame.

  • After that, log out and log in.

  • Then you will see the fcitx logo in the right control label. You can click it and from the menu to choose the Text Input method setting to add the google pinyin input method. Bravo, you finished it!
