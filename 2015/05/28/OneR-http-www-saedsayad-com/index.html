OneR

  • OneR是“一个规则”的缩写，是一个简单但是准确的分类算法，它对每个预测因子生成一个规则，然后选择使总误差达到最小的规则作为“一个规则”。要为预测因子创造一个规则，我们为每个预测因子相比目标变量构造了一个规则。实践表明，当产生更容易实现的规则时，OneR产生“一个规则”的准确率仅仅比最先进的分类算法产生规则小一点。

OneR算法

  • 对每个预测因子
    • 对每个预测因子的值，作出以下规则；
      • 统计当目标变量出现时，每个值出现的次数
      • 找出出现频数最多的类
      • 把规则赋给这个类对应的预测因子的值
    • 计算每个预测因子的规则产生的总误差值
  • 选择最小总误差的预测因子
  • 例子：根据相关的频数表，利用OneR找出使总误差最小的预测因子。

  • 简单地，根据频率表计算的总误差就是每个预测因子的贡献的衡量。总误差越低，预测因子的贡献率越高。
