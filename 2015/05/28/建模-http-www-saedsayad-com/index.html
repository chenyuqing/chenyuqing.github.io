建模-Modeling

  • 预测建模是创造预测结果的模型的过程。如果输出是分类类型的，我们称之为分类模型，如果输出是数值类型的，我们称之为回归模型。描述建模或者聚类模型是把相似的观测值归为同一个族群的过程。最后，关联规则可以在观测值中找到有趣的关联。
