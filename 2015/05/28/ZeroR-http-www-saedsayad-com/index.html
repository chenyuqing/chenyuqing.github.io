ZeroR

  • ZeroR是最简单的分类方法，它仅仅依赖输出变量而忽视所有的预测因子。ZeroR分类器简单地预测占大多数的分类(类别)。虽然ZeroR并没有可预测能力，但是它可以算出基线标准作为其他分类方法的基准。

算法

  • 构造一个关于目标变量的频率表，然后选择出现最频繁的值。
  • 例子：“Play Golf=Yes”是下面一个正确率为0.64的数据集的ZeroR模型。

预测因子的贡献

  • 没有预测因子贡献一说，因为ZeroR根本就没有用到它们(预测因子)。
