缺失值

  • 缺失值是一个比较常见的情况，而且你需要谨慎处理好它。数据中的一个缺失值可以表示很多东西。可能是数据不存在或者不适用或者没发生。也可能是输入数据的那个人不知道输入的数据正确与否，或者忘记输入了。不同的数据挖掘方法对缺失值有不同的处理方式。一般上，它们跳过缺失值，或者直接把包含缺失值的记录排除掉，或者利用均值来代替缺失值，或者利用现有的数据代替缺失值。

缺失值替代策略

  • 忽视带缺失值的记录。

  • 把缺失值换成一个全局常数

  • 根据领域知识手动填充缺失值

  • 如果是数值型的缺失值，换为它们的均值；如果是分类型的缺失值，换为它们出现最频繁的值。

  • 利用模型方法，譬如最近邻居，贝叶斯规则，决策树或者最大期望值算法解决。
