数值变量

  • 一个数值变量或者连续变量(或属性)表示在一个有限的区间或无限的区间中的任意一个数值(譬如，高度，重量，温度，血糖等等)。有两种类型的数值变量，区间和比率。一个区间变量拥有可解释的差值，但它没有要给真零点。一个比较好的例子是摄氏温度。区间变量上的数据可以进行加减，但是不能进行有意义的乘除。譬如，我们不能说某一天是另外一天的两倍热。相反，一个比率变量有真零点，它能够被加减乘除。(譬如，重量)。
