分类变量(Categorical Variables)

  • 一个分类变量或者离散形的变量是有两个或者多个分类的变量。常见的两类分类变量是名义性变量和有序变量。名义性变量对它的分类没有固定的排序。譬如，性别就是一个名义性变量，它包含男和女两个值，并且它们是没有排序可言的。有序变量是有一个明确的排序的。譬如，温度，带有3个有序分类(低，中，高)。频数表可以表示每个变量的所有分类值出现的次数。它可以通过加百分数来表示每个分类值出现的百分比。

  • 例子：带3个分类的房价变量(免费的，自己的，租来的)
