双变量分析(Bivariate Analysis)

  • 双变量分析是对两个变量(属性)同时分析。它探索两个变量之间关系的概念，两个变量之间是否存在某种关联，这种关联的强度如何，或者探索两个变量之间是否存在差异，这种差异的重要性等。有3中类型的双变量分析。
  1. 数值VS数值
  2. 分类VS分类
  3. 数值VS分类
