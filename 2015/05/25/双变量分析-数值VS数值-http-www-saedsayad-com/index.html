双变量分析-数值VS数值

散点图-Scatter Plot

  • 散点图是一种常用的两个数值变量之间的关系的可视化展示，而且它通常在我们找两个变量之间的线性相关或者拟合回归直线之前就画出来了。散点图可以明确的看出两个变量之间的关系(线性还是非线性)和它们关系的强度。在二维的散点图中可以加入更多信息，譬如，我们可以给点标记记号来表示第三个变量。如果我们正在处理的变量多于两个的数据集，一次展示所有双变量分析的方法是利用散点图矩阵。


线性相关(Linear Correlation)

  • 线性相关量化了两个数值变量之间的线性关系的强度。当两个数值变量之间没有关系时，当其中一个变量值改变(增加或者减少)时，另一个变量值没有变化趋势。

  • r仅仅表示两个变量之间的关系强度，它值在-1到1之间。-1代表完美线性负相关，1代表完美线性正相关，0代表没有线性相关。
