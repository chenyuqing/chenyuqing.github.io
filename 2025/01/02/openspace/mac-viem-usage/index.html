mac上编写viem例子的过程

  1. mkdir 01_test && cd 01_test
  2. npm init -y 初始化package.json文件
  3. npx tsc –init 初始化tsconfig.json文件
  4. touch client.ts编写代码
  5. npm install typescript ts-node viem dotenv types@node –save-dev
  6. 运行client.ts文件
    ts-node client.ts
